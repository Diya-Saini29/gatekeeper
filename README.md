# Gatekeeper - AI Cost Optimizer

## What Is This?

Gatekeeper is a smart router for AI systems that:
- Routes simple questions to cache (instant, free)
- Routes medium questions to small models (cheap)
- Routes complex questions to expensive models (best)

Result: **70% cost savings on AI bills**

## How It Works

1. User asks a question
2. Gatekeeper classifies the intent
3. Routes to appropriate system
4. Returns answer
5. Logs metrics to dashboard
